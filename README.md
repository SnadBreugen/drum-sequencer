# Drum Sequencer

Ein Step-Sequencer für Schlagzeuger mit automatischer Drum-Notation.

## Features

- 10 Drum-Stimmen: Crash, Ride, Open/Closed/Pedal Hi-Hat, drei Toms, Snare, Kick
- Regeln für echte Drummer: Hi-Hat-Varianten schließen sich aus, maximal zwei Toms gleichzeitig
- Vier Pattern-Slots (A/B/C/D), jeweils 1-4 Takte lang
- Takt-Kopieren, Pattern-Kette, echte Drum-Notation

## Starten

`index.html` im Browser öffnen — keine Installation nötig.
